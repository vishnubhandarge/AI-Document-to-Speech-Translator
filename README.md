# 🚀 AI Document to Speech Translator

A backend API built with **ASP.NET Core (.NET 10)** and **Azure AI services** that processes documents, translates content into multiple languages, and converts it into natural-sounding speech.

---

## ✨ Features

- 📄 Document text extraction  
- 🌍 Multi-language translation  
- 🔊 Neural text-to-speech  
- ⚡ Real-time processing (no storage)  
- 🧠 Clean service-based architecture  

---

## 🛠️ Tech Stack

- ASP.NET Core (.NET 10)  
- Azure AI Services (Document Intelligence, Translator, Speech)  
- REST API
